# 13. 动画生成（基于Remotion）

今天我们来用大名鼎鼎的remotion做做动画视频~做教程必备哦

我们来简单了解一下remotion：

Remotion 是一个开源的 React 框架，专门用于用代码（特别是 React 组件）程序化地创建真实 MP4 视频。它让开发者可以用熟悉的 React 语法、组件复用、CSS 动画、Canvas/SVG、三维等 web 技术来制作视频，而不是依赖传统视频编辑软件。
