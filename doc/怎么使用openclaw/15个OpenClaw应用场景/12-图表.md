核心用到的是这个库：excalidraw

**Excalidraw** 是一个开源的虚拟协作白板工具，以独特的**手绘风格（hand-drawn like）**闻名，让你能快速创建看起来像手绘的图表、流程图、线框图（wireframes）、思维导图或任何视觉想法。（PS，张咋啦那个直播白版就是借鉴这个项目……）

### 核心特点

- **手绘风格**：所有图形、线条和文字都带有自然的草图感，避免了传统工具的“冷冰冰”的完美主义，让创意更轻松、亲切。
- **无限画布**：没有画布大小限制，可以随意扩展，适合复杂项目或 brainstorm。
- **实时协作**：支持多人同时编辑（类似 Figma），并提供**端到端加密**，保护隐私。
- **免费开源**：完全免费使用，无需注册即可在浏览器中开始绘制。核心库支持 React 嵌入，可集成到自己的应用中。
- **其他实用功能**：
  - 深色模式、缩放/平移、撤销/重做。
  - 支持插入图片、形状库（Libraries）、本地化（多语言，包括中文）。
  - 导出为 PNG、SVG、JSON（.excalidraw 文件，可嵌入场景信息）。
  - 简单易用，学习曲线极低，常被誉为“绘图界的 Markdown”。

### 主要用途

- 快速原型设计（UI/UX 线框）。
- 技术架构图、系统设计图。
- 会议白板、脑暴、流程图。
- 个人笔记、教学演示（很多人在 Obsidian、VS Code 等工具中集成使用）。
- 团队协作沟通（特别适合开发者、设计师、产品经理）。
